CareerFinders

GitHub:

coming soon

About:

CareerFinders is designed to be a recommendation tool that utilizes input skill sets, personality traits, and job characteristics to suggest career fields or generalized roles that the user would be well-suited for. This application is intended for high school and college students to help them identify potential career paths that align with their interests and abilities.

Key Features:

  • Query mapping from raw text to skills using spaCy’s TextCategorizer and gensim’s Word2Vec
  • Efficient and accurate recommendation engine that matches input skills to careers
  • Career to reviews mapping that pulls in reviews related to the recommended careers

Personal Contributions:

  • Implemented the recommendation engine by:
    • Creating tf-idf matrices
    • Building an inverted index for skills
    • Developing numerous data structures to support the term-by-term cosine similarity algorithm
    • Mapping user input to the careers in our database
  • Implemented the recommendation algorithm, which accounts for the affinity scores each skill has in relation to the careers we have in our database.
  • Generated output rankings for the careers we recommend to each user, based on the computed similarity score for each career.

Tools Used:

Python, Flask, Docker, spaCy, gensim, numpy

ASL Translator App

About:

This project utilizes AI and ML for social good by creating an American Sign Language (ASL) translator app. Our aim is to improve communication between deaf and hearing individuals, promoting accessibility and breaking down barriers. The app has the potential to enhance accessibility in education, making it easier for students with hearing loss to participate and improving educational outcomes.

Key Features:

  • Fine-tuned Convolutional Neural Network (CNN) classifier using TensorFlow, Pandas, and Numpy
    • Achieved 99% accuracy on the Sign Language MNIST dataset
    • Demonstrates accurate recognition and categorization of sign language gestures
  • User-friendly UI for the ASL sign language translator app
  • Picture and video input functionality for seamless translation
  • Sign recognition quizzes and flashcards for interactive learning
  • Real-time translation capability for instant communication
  • Integration of the ML model directly into the app for efficient processing and response

Personal Contributions:

  • ML model integration into mobile app
  • Fine-tuned real-time translation from picture input
